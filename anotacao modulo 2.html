Input:
O comando "input" necessita de formulario para funcionar:
Que pode ser form method="POST"> /form> ou form method="GET"> /form>

1 - method="GET": Esse método envia os dados do formulário como parte da URL da requisição. Os dados são anexados na URL como parâmetros de consulta (query parameters). Esse é o método padrão quando nenhum valor é especificado para o atributo method. No entanto, é importante observar que o uso desse método pode limitar a quantidade de dados que pode ser enviada e também expõe os dados na URL, o que pode ser um problema de segurança se o formulário contiver informações sensíveis.

2 - method="POST": Esse método envia os dados do formulário no corpo da requisição HTTP. Os dados não são exibidos na URL, o que pode ser mais seguro para informações sensíveis. Além disso, o uso do método POST permite o envio de uma quantidade maior de dados em comparação ao método GET.

<.input type="search" name="search"/> - Para criar barra de "pesquisa"

<.input type="email" name="email"/> - Para criar barra de "email"

<.input type="submit" value="Enviar">

- Para enviar os Dados - Além de ser necessario criar o Form, tambem se faz necessario a criação do Botão para enviar as informações.

Vídeo:
<.video width="320" height="240" controls autoplay muted>
<.source src="./video/videoinformatica.mp4" type="video/mp4"/> <./video>

1 <.video width="320" height="240" : Esse comando informa o tamanho que o video terá no site.
2 Controls: Esse Mostrará os controles do vídeo.
3 autoplay muted : Esse dois comando é necessario para o vídeo iniciar automaticamente
4 O Autoplay é o comando para o video iniciar sozinho, porem é necessario que o video esteja mutado muted para rodar no Chrome por exemplo.

Audio

1 Esse caminha insere um audio normal no formato original.
<.audio src="caminho/do/arquivo-de-audio.mp3" controls><./audio>

2 Esse comando serve para adicionar diferentes tipos de formato de audio para compatibilidade com o navegador:
<.audio controls> <.source src="caminho/do/arquivo-de-audio.mp3" type="audio/mpeg"> <.source src="caminho/do/arquivo-de-audio.ogg" type="audio/ogg"> <.source src="caminho/do/arquivo-de-audio.wav" type="audio/wav"> <./audio>
