Anotações do Modulo 2

OBS: Remova o ponto antes de cada tag para que o codigo funcione

Input:

O comando "input" necessita de formulario para funcionar:
Que pode ser form method="POST"> /form> ou form method="GET"> /form>

method="GET": Esse método envia os dados do formulário como parte da URL da requisição. Os dados são anexados na URL como parâmetros de consulta (query parameters). Esse é o método padrão quando nenhum valor é especificado para o atributo method. No entanto, é importante observar que o uso desse método pode limitar a quantidade de dados que pode ser enviada e também expõe os dados na URL, o que pode ser um problema de segurança se o formulário contiver informações sensíveis.
method="POST": Esse método envia os dados do formulário no corpo da requisição HTTP. Os dados não são exibidos na URL, o que pode ser mais seguro para informações sensíveis. Além disso, o uso do método POST permite o envio de uma quantidade maior de dados em comparação ao método GET.

<.input type="search" name="search"/> - Para criar barra de "pesquisa"

<.input type="email" name="email"/> - Para criar barra de "email"

<.input type="submit" value="Enviar"> - Para enviar os Dados - Além de ser necessario criar o Form, tambem se faz necessario a criação do Botão para enviar as informações.
